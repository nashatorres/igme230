Interactive DMV Form

For my Interactive II course, we were assigned to create an interactive vehicle registration form from the existing NYS DMV that solves the lack of hierarchy and establishes clear organization, ultimately making filling out the form a more intuitive experience. I was the only designer on this project who carried out the universal UX design process. I did everything from research, user flows, wireframes, and iterations of my solution. To execute the design, I used Illustrator.



This was my very first time executing a UX process. I’m super grateful this project taught me how to do user flows, competitive analysis, and iterate designs. While I’m content with the outcome, it lacks the user personas, journey map, and deeply extensive research for me to declare it finished. I would like to revisit this project in the future as I grow into a more well-rounded UX designer.
